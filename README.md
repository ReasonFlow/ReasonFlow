# 🌟 ReasonFlow

ReasonFlow 是一个革命性的AI工作流设计工具，让开发新的AI应用变得前所未有的简单和高效。

## ✨ 特性

- 🎨 直观的可视化工作流编辑器
- 🧩 可扩展的节点系统
- 🚀 高性能Rust后端
- 💅 美观的Next.js前端
- 🔌 跨平台桌面应用支持

## 🛠️ 技术栈

- **后端**: Rust + Tauri
- **前端**: Next.js + React Flow + TypeScript
- **状态管理**: Zustand
- **样式**: Tailwind CSS

## 🚀 开始使用

1. 克隆仓库：
```bash
git clone https://github.com/ReasonFlow/ReasonFlow.git
cd ReasonFlow
```

2. 安装依赖：
```bash
# 安装前端依赖
cd frontend
npm install

# 安装Rust依赖
cd ..
cargo build
```

3. 运行开发服务器：
```bash
cargo tauri dev
```
